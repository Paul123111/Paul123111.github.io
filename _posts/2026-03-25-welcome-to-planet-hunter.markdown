<p>This is the first post for my final year project, <i>Planet Hunter</i>. <i>Planet Hunter</i> is a
 2D roguelite top-down twin stick shooter set in an uncharted space system. </p>

<p>Fight aliens on small randomly generated planets to take over these uncharted planets and collect new equipment/weapons.</p>

<p><i>Planet Hunter</i> procedurally generates planets using Perlin noise and cellular automata. It also uses behaviour trees for smart enemy AI. There is also co-op multiplayer that implements Kubernetes to orchestrate game rooms players can join.</p>
